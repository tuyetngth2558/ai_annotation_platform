# Docs Index

Thư mục `docs/` là nơi tập trung tài liệu theo nhóm chủ đề.

## Cấu trúc đề xuất

- `00_project_management/`
  - tracking tiến độ
  - task export/import từ ClickUp
- `01_prd/`
  - PRD gốc
- `02_scope_planning/`
  - scope MVP
  - phạm vi build / hoãn
- `03_ba/`
  - BA theo người hoặc theo chủ đề
- `04_diagrams/`
  - workflow
  - context diagram
  - architecture diagrams
- `05_architecture/`
  - technical design
  - ERD
  - API contracts
- `06_test_qa/`
  - test plan
  - checklist
  - bug flow
- `07_devops/`
  - deployment
  - env setup

## Trạng thái hiện tại

Hiện một số tài liệu vẫn đang nằm ở root để tránh làm gián đoạn file hiện có. Team có thể chuyển dần vào `docs/` khi bắt đầu ổn định repository.
