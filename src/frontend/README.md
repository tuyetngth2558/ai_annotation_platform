# Frontend

| Thư mục | Trạng thái |
|---|---|
| `web/` | ⭐ App React + Vite + TypeScript thật (feature-based co-locate). Code ở đây. |
| `prototype/` | 4 HTML vanilla — **tham chiếu UX**, không build tiếp. |

## Chạy / phát triển
Xem [docs/onboarding/frontend-dev.md](../../docs/onboarding/frontend-dev.md).

Cấu trúc `web/src/`: `app/` (router/layout/provider) · `features/<feature>/`
(pages/components/locales) · `shared/` (ui/lib/hooks/types) · `i18n/` · `styles/` (OKLCH theme).
